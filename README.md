# seekh
A simple Course Management System developed using the following technologies :

1 - Express.js <br />
2 - Typescript <br />
3 - Prisma - Database Client <br />
4 - SQLite <br />
5 - Next.js <br />
6 - Tailwind CSS <br />
7 - Jest - the testing tool <br />
8 - Swagger - API documentation tool <br />
